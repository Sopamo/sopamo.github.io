Website deploy

Complete the following steps before/directly after launching a new website

  • Create google analytics entry (with ga('set', 'anonymizeIp', true); for german websites)
  • Create the Google Webmaster Tools account (with sitemap)
  • Double check robots.txt
  • Check title and description tags for all subpages
  • Check if the 4xx and 5xx pages are customized and fancy
  • Setup proper error reporting (Via email or Bugsnag)
  • Run http://www.webpagetest.org/ with the most critical pages (especially landing pages)
  • Run https://developers.google.com/speed/pagespeed/insights/ and fix everything in scope
  • Create and embed a favicon and app colors http://realfavicongenerator.net/
  • For german sites: Create the imprint page. Is it reachable from all sites with a maximum of 2 clicks?
  • Double check all sites with a desktop, tablet and mobile device
  • Setup MySQL and Data backups
  • Enter the company in Google Places
  • Create a reference page on the sopamo.de website
